# Sistema de Gestión para Lashista

## Descripción
Este proyecto consiste en el desarrollo de un sistema para una lashista que permite administrar de forma sencilla y ordenada el registro de clientes, la agenda de citas, el control de pagos y el historial de servicios realizados. La aplicación busca mejorar la organización del trabajo diario, optimizar el tiempo y brindar un mejor servicio a los clientes.

## Motivación
La motivación de este proyecto es aplicar los conocimientos adquiridos en programación para desarrollar una herramienta práctica que resuelva una necesidad real, facilitando la gestión del negocio de una lashista y ayudando a mantener un mejor control de la información.
